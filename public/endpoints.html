Canvas Export API Endpoints

The POST requests must be done through postmen

1. Create Canvas

Endpoint: /api/create-canvas

Method: POST

Description: Use this endpoint to create a new canvas. Provide canvas data in the request body.

2. Get Canvas Data

Endpoint: /api/get-canvas

Method: GET

Description: Retrieve the canvas data created earlier.

3. Get Elements Data

Endpoint: /api/elements

Method: GET

Description: Retrieve the canvas elements created

4. Add Circle

Endpoint: /api/add-circle

Method: POST

Description: Add a circle to the canvas. Provide circle parameters in the request body.

5. Add Text

Endpoint: /api/add-text

Method: POST

Description: Add text to the canvas. Provide text parameters in the request body.

6. Add Line

Endpoint: /api/add-line

Method: POST

Description: Add a line to the canvas. Provide line parameters in the request body.

7. Add Rectangle

Endpoint: /api/add-rectangle

Method: POST

Description: Add a rectangle to the canvas. Provide rectangle parameters in the request body.

8. Add Triangle

Endpoint: /api/add-triangle

Method: POST

Description: Add a triangle to the canvas. Provide triangle parameters in the request body.

9. Export Canvas (POST)

Endpoint: /api/export

Method: POST

Description: Export the canvas as an HTML file. Use this endpoint first to prepare the export data.

10. Export Canvas (GET)

Endpoint: /api/export

Method: GET

Description: Download the exported HTML file.
